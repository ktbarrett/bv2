# BV2

Potential replacement for `BinaryValue` as seen in cocotb.

## Installation

Get the source.

```
git clone https://github.com/ktbarrett/bv2
```

Install with `pip`.

```
pip install bv2
```

## Running tests

Get `nox`, if you don't already have it.

```
pip install nox
```

Run regression with `nox`.

```
nox -f bv2/noxfile.py
```
